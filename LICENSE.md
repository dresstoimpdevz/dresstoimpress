## 🔧 Troubleshooting & Common Issues

### ❌ Script Won't Execute

**Symptoms:** Executor shows no output, GUI never appears.

**Fix:**
- Ensure the game is fully loaded before attaching
- Run your executor with Administrator privileges
- Verify you're using the correct `.lua` file from the `/scripts` folder

### ❌ Auto Win Not Triggering

**Symptoms:** Round ends without the auto-submit firing.

**Fix:**
- Check that `auto_win_delay` is not set higher than the round timer
- Confirm `outfit_theme_match = true` in your config
- Reload the script mid-session via the GUI's "Reinject" button

### ❌ VIP Features Not Showing

**Symptoms:** VIP UI elements don't appear after enabling simulation.

**Fix:**
- Set `vip_simulate = true` in config **before** attaching to the game
- Rejoin the session after enabling the toggle — some features require a fresh load

### ❌ Crash on Android

**Symptoms:** Game closes unexpectedly after injection.

**Fix:**
- Reduce `troll_interval` to `5.0` or higher on mobile
- Disable the outfit AI matching on lower-end devices
- Ensure you have at least 2 GB of free RAM before launching

### ❌ macOS Permission Denied
